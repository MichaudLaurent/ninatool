NINA - Nonlinear Inductive Network Analyzer

Welcome to the NINA tool! Visit the NINA GitHub repo to try our Python package to analyze the local properties of an arbitrary flux-biased Josephson loop.

The theory behind the NINA package can be found in the paper Hamiltonian extrema of an arbitrary flux-biased Josephson circuit.

A YouTube seminar on the theory behind NINA can be found at the IBM Qiskit seminar series #114.
